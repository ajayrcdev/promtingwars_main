# ⚔️ Prompting Wars

A fun prompt battle game — vote on which AI prompt is better!

## Deploy

### Vercel (easiest)

```bash
npm i -g vercel
vercel --prod
```

Or just connect your GitHub repo to [vercel.com/new](https://vercel.com/new).

### Netlify

Connect your repo at [netlify.com](https://netlify.com) — it auto-detects Vite.

## Develop

```bash
npm install
npm run dev
```
